
前提環境

OS

macOS（Apple Silicon / Intel）

Windows 10 以降

Python

Python 3.10〜3.12 を推奨

必要ツール

git（任意。ZIP 配布の場合は不要）

コマンドプロンプト / PowerShell（Windows）、ターミナル（Mac）

本アプリケーションは Streamlit を用いた Web アプリです。

## ✔ 動作確認済み環境

本アプリは以下の環境で動作確認済みです：

- Python 3.11
- streamlit==1.40.2
- macOS / Windows 11

必要なライブラリは Streamlit のみです。

1-1. ソースコードの配置

任意の作業ディレクトリを用意します（例：~/projects/communication-support-app）。

以下のファイル群一式を同じフォルダに配置します。

main.py

initialize.py

components.py

constants.py

utils.py

README.md

GitHub リポジトリとして配布する場合は、このフォルダがそのままリポジトリのルート階層になります。

1-2. 仮想環境の作成と依存パッケージのインストール
■ macOS の場合
# プロジェクトフォルダへ移動
cd /path/to/your/project

# 仮想環境を作成
python3 -m venv env

# 仮想環境を有効化
source env/bin/activate

■ Windows の場合（PowerShell 例）
# プロジェクトフォルダへ移動
cd C:\path\to\your\project

# 仮想環境を作成
python -m venv env

# 仮想環境を有効化
.\env\Scripts\activate


1-3. アプリケーションの起動

仮想環境を有効にした状態で、以下を実行します。

streamlit run main.py


実行後、ブラウザが自動的に立ち上がり、アプリ画面が表示されます。

自動で開かれない場合は、ターミナルに表示される URL（http://localhost:8501 など）をブラウザにコピー＆ペーストしてください。

1-4. アプリの停止方法

ターミナル / PowerShell 上で Ctrl + C を押下するとサーバーが停止します。

仮想環境を終了する場合は、以下を実行します。

# macOS:

deactivate


# Windows:

deactivate
