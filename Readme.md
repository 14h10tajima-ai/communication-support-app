# コミュニケーション補助アプリ（ビジネス版）
価値観共有・アイスブレイク・チーム内対話の円滑化を目的とした  
ビジネス向けコミュニケーション支援アプリです。  
Python + Streamlit にて構築されています。

---

# 🧭 概要

本アプリは、  
**構造化された対話プロンプト** および  
**ロール（役割）付与による会話モード** を通じて、  
心理的安全性の高いコミュニケーション環境を実現するツールです。

ビジネスシーンにおいて自然に発話しやすく、  
「本音を引き出しやすい」設計を採用しています。

主な特徴：

- **アーサー・アーロン研究に着想を得た対話モード**  
  ※質問文は原文を使用せず、  
    **独自翻訳・意訳・再構成したオリジナル文章**
- **価値観共有・合意形成を促すビジネス向けオリジナルモード**
- **ロール付与（キャラクター設定）により、  
  発言と“本人評価”が切り離される安全な仕組み**

日本的組織文化（調和・合意形成・遠慮）に配慮し、  
対話のハードルを下げるコミュニケーションフローを実現しています。

---

# 🎯 開発背景

近年、AI によるキャラクターチャットやロールプレイ型アプリは多数存在します。  
しかし **対面コミュニケーション** を支援する実用的ツールは依然として少ない状況です。

特に日本では：

- 個性の強い主張を避ける傾向  
- 調整・合意形成を重視する文化  
- 自己評価・他者評価に敏感  

といった特徴があり、海外の  
**“個性を発揮する前提” のコミュニケーションツールは噛み合いにくい**  
という課題があります。

そのため本アプリでは：

- **ロール付与** により、自己評価から解放された発話を可能にする  
- **価値観の違いを安全に共有できる質問設計**  
- **ビジネス場面での対話を阻害しない、シンプルで中立的な UI**

という方向性を採用しています。

# 🧩 機能一覧

## ■ 1. アーサー・アーロン着想モード（独自翻訳版）
- 原研究の「段階的に深まる構造」を参考に設計  
- 質問文はすべて **独自翻訳・意訳・再構成したオリジナル**

## ■ 2. ビジネス価値観共有モード（完全オリジナル）
- 業務スタイル・優先順位・意思決定の癖など  
  “仕事に直結する会話テーマ” に最適化

## ■ 3. ロール付与モード
- キャラクターや属性を付与して回答してもらうことで  
  **発言が“自分そのもの”と評価されず、安心して話しやすい**

## ■ 4. 重複防止ロジック
- モード × 難易度ごとに **未使用のお題のみ** を抽出

## ■ 5. シンプルな Streamlit UI
- ボタン操作のみで直感的に利用可能
